Section 04

What makes this unique

01
Two honest experiments
With-grades R²=0.975 vs without-grades R²=0.374. The gap tells the real story — prior performance dominates. Most projects report one number.
02
Feature engineering
study_fail = studytime/(failures+1) — effectiveness not just effort. grade_trend = G2−G1 — academic momentum. Each feature has a hypothesis behind it.
03
5-fold cross-validation
CV R²=0.964 vs test R²=0.975 — confirms the model generalises. Not a lucky 80/20 split. Every model evaluated rigorously.
04
Dropouts handled
G3=0 students are real dropouts, not noise. Exp A excludes them for regression. Exp B includes them for enrollment-time realism.
05
Fully dynamic dashboard
Every metric, chart and KPI on this page is fetched live from Flask at startup. Retrain → refresh → everything updates. Zero hardcoded values.
06
9-endpoint REST API
/predict, /metrics, /features/<mode>, /dataset/stats, /dataset/corr, /dataset/grades, /dataset/scatter, /health, /api.
